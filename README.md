# Rime 鼠须管输入法 for Mac 配置方案
## 效果
![](图片/效果图.jpg)

1. 前往 [Rime](https://rime.im/) 下载并安装好输入法。
2. 将【字体】文件夹内的 2 个文件安装到 Mac【字体册】。
3. 菜单栏【前往】>【前往文件夹…】输入 `~/Library/Rime`，打开 Rime 目录文件。
4. 将【配置文件】里所有文件粘贴覆盖到 Rime 目录文件夹内。
5. 点击【重新部署】完成。

>注：按 `Control + ~` 选择 `朙月拼音`，可以切换输入法方案。
